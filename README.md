# 📊 Insurance Claims EDA Dashboard

A comprehensive exploratory data analysis project for insurance claims fraud detection, built with Python and Streamlit.

## 🎯 Project Overview

This interactive dashboard analyzes 1,000 insurance claims to identify fraud patterns and business insights. The project demonstrates end-to-end data analysis skills from data loading and cleaning to advanced visualizations and actionable business recommendations.

## 🛠️ Technologies Used

- **Python** - Core programming language
- **Pandas** - Data manipulation and analysis
- **Streamlit** - Interactive web application framework
- **Plotly** - Advanced data visualizations
- **CSS** - Custom styling for professional presentation

## 📊 Key Features

### Data Analysis
- Comprehensive fraud pattern identification
- Statistical analysis of claim amounts and distributions
- Correlation analysis between key variables
- Demographic analysis of policyholders

### Visualizations
- Interactive histograms and box plots
- Scatter plots for relationship analysis
- Correlation matrix heatmap
- Professional metric cards with custom styling

### Business Insights
- Fraud rate analysis (24.7% of claims)
- High-risk demographic identification
- Claim amount patterns by fraud status
- Strategic recommendations for fraud prevention

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- Required packages: `streamlit`, `pandas`, `plotly`

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd insurance-claims-eda
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
streamlit run app.py
```

## 📈 Key Findings

- **Fraud Rate:** 24.7% of all insurance claims are fraudulent
- **Claim Values:** Fraudulent claims average $60,000 vs $50,000 for legitimate claims
- **High-Risk Groups:** Policyholders over 50 show 30.7% fraud rate
- **Common Incidents:** Multi-vehicle collisions are most frequent
- **Correlations:** Strong relationship between age and customer tenure (0.92 correlation)
